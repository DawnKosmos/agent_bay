# agent_bay/libs/go

Shared Go libraries for agent_bay projects.

## Packages

- **cdc** — PostgreSQL Change Data Capture (CDC) using logical replication for real-time data streaming.
- **gogen** — Go struct to TypeScript interface generator for type-safe API contracts.
- **ts_util** — TypeSense schema generator that extracts field types and options from Go struct tags (e.g., `index`, `facet`).
- **uuid** — UUID wrapper with nullable support and database driver integration.
- **wrap/fiber** — Type-safe Fiber HTTP handler wrappers for simplified route definitions.
